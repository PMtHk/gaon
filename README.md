# gaoncare-homepage

가온방문요양센터 홈페이지 개발

<br/>

## 서비스

**1. 방문요양 제도 및 서비스 소개**

**2. 공지사항 등록 및 수정**

**3. 상담 신청 및 완료 처리**

**4. 상담 신청 시 관리자에게 SENS로 문자 발송**

- 개발 당시, 네이버의 SENS의 새 앱 등록이 막혀 Slack API를 활용하여 알림 발송으로 대체

<br/>

## 프로젝트 목표

**1. Next.js 연습 및 실제 서비스 배포까지 구현**

- Next.js 14 를 이용한 구현
- AWS Amplify 를 이용한 배포

**2. 검색엔진 최적화(SEO)**

- sitemap.xml, robots.txt 작성
- static 및 dynamic metadata 작성
